# كود زائف بسيط للموقع (بالعربي)

هذا مثال بسيط جدًا للكود الزائف باللغة العربية، وبأسلوب:
"إذا المستخدم... إذا نعم... إذا لا... وإلا..."

## 1) كود زائف للتنقل داخل الموقع

```
ابدأ
  إذا المستخدم ضغط على "الخدمات"
    اذهب إلى صفحة الخدمات
  وإلا إذا المستخدم ضغط على "الباقات"
    اذهب إلى صفحة الباقات
  وإلا إذا المستخدم ضغط على "تواصل معنا"
    اذهب إلى صفحة التواصل
  وإلا
    ابقَ في الصفحة الرئيسية
نهاية
```

## 2) كود زائف للتحقق من نموذج التواصل

```
ابدأ
  تحقق من أن الاسم والبريد والرسالة مكتوبة

  إذا كانت الحقول المطلوبة مكتملة
    تحقق من صحة صيغة البريد

    إذا كان البريد صحيحًا
      أرسل النموذج
      اعرض رسالة: "تم الإرسال بنجاح"
    وإلا
      اعرض رسالة: "صيغة البريد غير صحيحة"
  وإلا
    اعرض رسالة: "يرجى تعبئة الحقول المطلوبة"
نهاية
```

## 3) كود زائف لاختيار الباقة

```
ابدأ
  إذا المستخدم اختار باقة
    احفظ اسم الباقة المختارة
    انقله إلى صفحة التواصل
  وإلا
    اعرض رسالة: "اختر باقة أولًا"
نهاية
```
